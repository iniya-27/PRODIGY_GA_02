
# 🖼️ Stable Diffusion for Text-to-Image Generation — Complete Explanation

---

## 1. What is Stable Diffusion?

Stable Diffusion is a deep learning model that generates images from text prompts.

- It is a text-to-image generative model
- It converts random noise into images step-by-step
- It understands natural language prompts like "a sunset over the ocean"

---

## 2. What is the Diffusers Library?

The Hugging Face Diffusers library provides tools to run diffusion models easily.

It includes:
- Pretrained models
- Image generation pipelines
- Control over inference steps, guidance scale, and schedulers

---

## 3. Model Used

We use Stable Diffusion v1.5.

Why:
- Lightweight compared to newer models
- High-quality outputs
- Widely used for learning and experimentation

## 5. How Stable Diffusion Works

Text Prompt → Encoding → Random Noise → Denoising Steps → Final Image

Core idea:
- Start with noise
- Gradually refine it
- End with a clear image matching the prompt

---

## 6. Image Generation Function

## 7. Experiments

### Experiment 1: Text to Image
Multiple prompts tested:
- sunset over ocean
- cyberpunk city
- magical forest
- robot in library

### Experiment 2: Guidance Scale
Controls how strictly the model follows the prompt.

Low → creative  
High → strict output  

### Experiment 3: Inference Steps
More steps = better quality but slower generation.

### Experiment 4: Negative Prompts
Helps remove unwanted outputs like blur or distortion.

### Experiment 5: Art Styles
Same subject generated in different styles:
- oil painting

- pixel art
